# CapstoneDatabricksDemo

This repository is for the capstone course demonstrating how to use Databricks Free/Community Edition to build a simple ETL pipeline with volume, bronze, silver, and gold tables.

## Purpose

This walkthrough will guide you through:
- Signing up for Databricks Free/Community Edition
- Connecting Databricks to GitHub and cloning this repository
- Building a simple ETL pipeline using Databricks, including volume, bronze, silver, and gold tables

## Getting Started

### 1. Sign Up for Databricks Free/Community Edition

1. Go to [https://community.cloud.databricks.com/](https://community.cloud.databricks.com/)
2. Click **Get Started for Free** and follow the prompts to create your account.

### 2. Connect Databricks to GitHub and Clone This Repo

1. In your Databricks workspace, click on the **Repos** icon in the sidebar.
2. Click **Add Repo** > **Clone remote Git repo**.
3. Enter the URL of this repository:  
   `https://github.com/<your-org>/CapstoneDatabricksDemo.git`
4. Click **Create**.

### 3. What You'll Build

- **Volume Table:** Raw data ingestion
- **Bronze Table:** Cleaned and structured raw data
- **Silver Table:** Refined data with business logic applied
- **Gold Table:** Aggregated data ready for analytics

Follow the notebooks in this repo for step-by-step instructions.
